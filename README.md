# test task for AXSYS

Реалізувати наступну логіку:
Створити апку в якій буде акордеон, перша сторінка це список сервісів, друга це список провайдерів. При виборі сервіса вкладка переключаєтсья на провайдерів цього сервіса, після вибору сервісу та провайдера активізується кнопка `створити івент`. Після настискання якої нас переключає на скрін з фінальним повідомленням - `You just selected SERVICE_NAME with PROVIDER_NAME. Total price will be - PRICE, Duration DURATION minutes`. А також кнопка яка пропонує повторити дію ще раз (добто повернути все до початквого виду і виконати цей самий флов ще раз).
Вимоги:
1. Логіка має бути реалізована на react + redux (без різниці з яким стартовим пакетом чи `create-react-app` чи будь яким іншим).
2. Не заморочуватись з дизайном, реалізувати найпростіший з можливих, при бажанні використати bootstrap або іншу UI бібліотеку. суто функціонал але щоб мало мінімальний естетичний вигляд.
3. Без препроцесорів і постпроцесорів.
4. Код залити на gitHub
